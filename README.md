**Web-Based Ordering System for Ry's De Cocina**

**DESCRITION**

The Web-Based Ordering System for Ry’s De Cocina is a prepaid online ordering platform designed to address the issue of unclaimed orders at a small food stall in Bulan, Sorsogon. The system requires customers to pay in advance via GCash and upload proof of payment before their order is prepared. This ensures only verified and paid orders are processed, reducing food waste and improving operational efficiency.


**Key Features:****

1. Online menu browsing and ordering

2. Manual GCash payment verification

3. Admin dashboard for order, menu, and user management

4. Email notifications for order status updates

5. Customer loyalty points system

6. Sales reporting and transaction tracking

7. Responsive design for desktop and mobile



**Summary of the System’s Purpose and Features
Purpose:**

To eliminate unclaimed orders and food waste by implementing a prepaid, web-based ordering system that ensures payment before food preparation.

**Main Features:**

**Customer Side:****

User registration and login

Digital menu with real-time availability

Cart and order placement

GCash payment proof upload

Order history and status tracking

Loyalty points earning and redemption

Admin Side:

Dashboard for order verification

Menu management (add, edit, disable items)

User and staff account management

Sales reports and transaction records

Email notification management

Technical Highlights:

Built with OutSystems (low-code)

Uses MySQL for database

Hosted on Apache (XAMPP for local, cloud for production)
